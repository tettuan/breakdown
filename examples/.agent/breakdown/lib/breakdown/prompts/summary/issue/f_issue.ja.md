# イシュー要約テンプレート

## 入力
- {input_text}

## 出力
- {destination_path}

---

## 要約観点
- イシューの概要
- 主要な要件・課題
- 依存関係の要約
- 優先度・緊急度
- 見積もり工数
- 状態・ブロッカー

## 手順
1. 入力情報を上記観点で要約してください。
2. 100文字、200文字、300文字の単位で要約を作成してください。
3. 章立てし、段落を構成してください。
4. 不足情報があれば「不足情報」として明記してください。
5. テンプレート変数は必ず利用してください。

## 出力形式
- 構造化されたMarkdownで出力してください。
